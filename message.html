Download Not Allowed

Your last use of this mail server did not produce a web page. This could have happened for these reasons.

  • You attempted to download a binary file.
  • The web server did not deliver a web page within the allowed time.

Downloading of binary files is not allowed due to the high level of abuse that was encountered in the past. People were downloading videos, MP3 files, porn and other things that just ate up bandwidth and often bounced when it hit the target mail server. Also, some servers use odd technologies or Shockwave Flash to inspect the visitor's web browser. Those are binary files as well and won't get downloaded.

Speed is of the essence with this tool. Each request has a reasonable limit before the connection to the other web server is killed. This helps ensure that slow web servers don't kill this service for everyone.

This webpage retrieval tool's purpose is to help people view the web for research and to gain new knowledge.

If you feel that you received this message in error, I suggest you simply try the request again at a different time. Perhaps the web server was slow temporarily or they had issues delivering the web page to you.
